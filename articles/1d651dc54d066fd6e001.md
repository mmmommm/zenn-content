# 画面の写り

まずはこれです、私はゼミなどでも使うのですが逆光になっていたりして全く顔が見えないことがあります。
大切なのは二点
- 後ろから照らさないこと(逆光になり顔が見えなくなる)
- 顔に光を当てること

[こちらの記事](https://note.com/kengo_25/n/n3ee301e84834)が非常にわかりやすいです。
私はzoomを使用するときは正面からスタンドライトを顔に当てています。最初は眩しいですが、かなり違って見えるのでおすすめです！！！
もしない場合は100均でタッチライトが売っているので100円ぐらいは出して買いましょう!

# チャット

イベントなどでチャットを送ることがあるかと思いますが、宛先に注意しましょう！
イベントなどでは最初が`すべてのパネリスト`になってしまっているので`皆様`に変更しましょう

# 画面共有
画面共有ができないときはmacの場合
- システム環境設定
- セキュリティとプライバシー
- 画面収録を選択して鍵をクリック

で変更できます。

# 招待
![](https://storage.googleapis.com/zenn-user-upload/qx6eckdfb4kkxva6bfxqsptym4zc)
参加者　→　招待から招待リンクを作成できます。
`招待リンクをコピー`だとパスワード不要の招待リンクをコピーできます。
`招待のコピー`だとパスワードが必要な招待リンクをコピーできます、パスワードは右下の文字列です。

![](https://storage.googleapis.com/zenn-user-upload/yru052ec5zfukgcc11rj901oeid4)
参加者の右下の詳細から様々な設定ができ
`待機室を有効化`のチェックを外すとURLを踏んだ人がホストの承諾なしでそのままzoomに参加できます。

# zoomミーティングに参加できないとき
参加できない原因として考えられるのは
1. ホストが参加していない
2. urlもしくはミーティングIDが間違っている

1の場合
- 開始時間より前なのであれば少し待ちましょう
- 他のミーティングIDで開始している場合があるのでホストにミーティングIDを確認してもらってください。
2の場合
- URLを踏んでもダメでもミーティングIDを打ったら入れたことがあるので入れないときは試してみるといいと思います。
- 友人が参加できているのに自分が参加できない場合はミーティングに入っている人は左上の`i`のマークからミーティング情報が確認できるのでミーティングIDなどを教えてもらえば参加できます。
